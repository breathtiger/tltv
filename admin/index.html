Apps Script 後端 URL

後台儲存功能需要 Google Apps Script Web App。URL 僅存於本瀏覽器 localStorage，不會上傳至任何伺服器。

Google Drive 圖片資料夾

上傳輪播圖、消息封面圖、主持人照片至此資料夾，再將檔案 ID 貼入對應欄位。
取得 ID 方式：在 Drive 中對圖片按右鍵 → 共用 → 複製連結，URL 中 /d/ 與 /view 之間的那段即為 ID。

開啟 Drive 圖片資料夾
Apps Script 部署說明
  1. 前往 script.google.com，新增空白專案
  2. 將 admin/code.gs 的全部內容貼入編輯器，儲存
  3. 執行 setupAdminPassword 函式一次（設定管理員密碼）
  4. 部署 → 新增部署作業 → 網頁應用程式；執行身分：我；存取權：所有人
  5. 複製部署 URL，貼到上方欄位儲存
  6. 主持人管理首次使用時，系統會自動在試算表建立 hosts 工作表
